# Luminosity CLI

Luminosity CLI (`@gravito/luminosity-cli`) provides command-line tools to manage sitemap generation and maintenance workflows.

## Installation

```bash
bun add @gravito/luminosity-cli
```

## Common Commands

```bash
# Generate sitemap files
gravito-seo generate

# Flush cache
gravito-seo flush
```

## Usage Notes

- Use this CLI when you need manual control or CI integration.
- Pair it with Luminosity configuration for consistent output.
