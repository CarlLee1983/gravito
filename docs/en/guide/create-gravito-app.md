# Create Gravito App

`create-gravito-app` is a thin wrapper around the Gravito CLI. It bootstraps new projects with curated templates.

## Usage

```bash
bunx create-gravito-app@latest my-app
```

## What it does

- Invokes the Gravito CLI `create` workflow
- Generates a starter project with recommended structure
