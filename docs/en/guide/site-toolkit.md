# Site Toolkit

Site Toolkit (`@gravito/site`) is a static documentation toolkit built on Monolith, Cosmos, and Constellation. It is used to generate docs sites with localization and sitemaps.

## Installation

```bash
bun add @gravito/site
```

## Basic Usage

```bash
# Build the runtime
bun run build

# Generate static output
bun run generate
```

## What it provides

- Markdown-driven content via Monolith
- Locale handling via Cosmos
- Sitemap generation via Constellation
