# Site Toolkit

Site Toolkit（`@gravito/site`）是以 Monolith、Cosmos 與 Constellation 組成的靜態文件工具，可用於多語系文件站。

## 安裝

```bash
bun add @gravito/site
```

## 基本使用

```bash
# 建置執行檔
bun run build

# 生成靜態輸出
bun run generate
```

## 提供功能

- Monolith 驅動的 Markdown 內容
- Cosmos 多語系管理
- Constellation sitemap 產生
